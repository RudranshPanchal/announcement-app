# Shopify Announcement App

## Features

- Shopify Embedded App
- MongoDB Storage
- Shopify Metafield Sync
- Theme App Extension
- Announcement Banner

## Tech Stack

- React Router
- Node.js
- MongoDB
- Mongoose
- Shopify Admin GraphQL
- Theme App Extension

## Installation

npm install

## Environment Variables

SHOPIFY_API_KEY=
SHOPIFY_API_SECRET=
SHOPIFY_APP_URL=
SCOPES=
MONGODB_URI=

## Run

shopify app dev
